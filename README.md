# 🐍 SnakeX

A classic browser-based Snake Game built using **HTML, CSS, and JavaScript**.

### ✨ Features
- 🎮 Smooth and responsive gameplay
- 🍎 Random food generation
- 📊 Real-time score tracking
- ⏱️ Built-in game timer
- 🔄 One-click restart functionality
- 🏆 Persistent high-score storage using Local Storage
- 📱 Responsive and clean user interface

Enjoy the timeless Snake experience with modern web technologies—play instantly in your browser!
